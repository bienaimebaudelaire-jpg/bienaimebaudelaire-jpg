🧠 GUIDE DE DÉMARRAGE HUMEAN

1 DÉMARRER LE SERVEUR

Double-cliquez sur ce fichier :

🚀 START_SERVER.bat

Une fenêtre noire va s'ouvrir - NE LA FERMEZ PAS !

✅ Vous devriez voir :
"🧠 HUMEAN v2.0"
"Démarrage du serveur cognitif..."
"Serveur démarré sur http://localhost:5000"

2 OUVRIR LE LAUNCH CENTER

Double-cliquez sur ce fichier :

🎯 LAUNCH_CENTER.html

Le statut devrait passer à EN LIGNE

3 UTILISER HUMEAN

Depuis le Launch Center, vous pouvez :

📊 Dashboard Principal 🎨 Dashboard Complet 🎛️ Panneau Contrôle

❓ EN CAS DE PROBLÈME

Le serveur ne démarre pas ?

  1. Vérifiez que Python est installé : ouvrez un terminal et tapez python --version
  2. Installez les dépendances : pip install flask requests
  3. Redémarrez l'ordinateur
  4. Relancez START_SERVER.bat

🎯 RÉSUMÉ RAPIDE

Pour utiliser HUMEAN rapidement :

  1. Double-clic sur START_SERVER.bat
  2. Double-clic sur LAUNCH_CENTER.html
  3. Cliquez sur "📊 DASHBOARD"
  4. Explorez ! 🚀
